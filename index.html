GitHub Project

Demonstrate git workflow

  1. Create a GitHub account
  2. Generate SSH key pairs and add authentication
  3. Create a repository in remote
  4. Create Integration branch and Hotfix branch based off of master
  5. Create feature1 and feature2 based off of Integration branch
  6. Clone feature2 repository to local
  7. Add changes to local
  8. Stage the changes and commit to local. Push the changes to remote
  9. Switch to feature1 branch in local
